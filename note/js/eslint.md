# 使用eslint代码校验

## 快速开始

### 安装&开始使用

```sh
# 项目安装eslint
npm install --save-dev eslint

# 项目中使用eslint代码校验
./node_modules/.bin/eslint .js src/

# 项目中使用eslint代码修复
./node_modules/.bin/eslint --fix  .js src/

```
